# **App Name**: SyncTask Pro

## Core Features:

- Task Listing: Display a real-time list of all tasks retrieved from the Firebase database.
- Task Creation: Allow users to add new tasks to the Firebase database through a simple input field.
- Task Completion Toggle: Enable users to mark tasks as complete or incomplete, updating their status in Firebase.
- Task Deletion: Provide a way for users to remove tasks from their list, deleting them from Firebase.
- Real-time Synchronization: Automatically update the task list on the page whenever data changes in the Firebase database.

## Style Guidelines:

- The app uses a light color scheme, suitable for productivity. The primary color is a modern blue-purple (#4D4DEF), embodying reliability and focus. The background color is a subtly tinted light grey (#F2F3F7), maintaining a clean canvas for content. A vibrant cyan (#19B3E6) serves as the accent color for interactive elements and highlights, ensuring high visibility.
- The 'Inter' font (sans-serif) is recommended for all text. Its objective, modern aesthetic ensures high readability for task lists and input fields, aligning with a focused and efficient user experience.
- Utilize a consistent set of clean, outline-style icons. These will clearly represent actions like add, complete/uncomplete, and delete, without distracting from the main content.
- The layout focuses on a single-page, minimalist design with clear content hierarchy. Tasks should be prominently displayed, and input controls should be intuitive and easily accessible, optimizing for efficiency on all screen sizes.
- Incorporate subtle and quick animations for task interactions, such as fading in new tasks or briefly highlighting updated items, to provide smooth visual feedback without being intrusive.
